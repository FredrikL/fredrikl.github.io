--- layout: post title: ttTumblr categories: - Code tags: - Android - ttTumblr - tumblr status: publish type: post published: true meta: _edit_last: '2' tweetlyUpdater_bitlyUrl: http://bit.ly/bFbM2E --- A couple of days ago I uploaded the first version of ttTumblr on Android Market. It's a small app that allows the user to create content on a tumblr blog. At the moment, release 3, supports posting images from both gallery and by taking an image with the camera, creating text, quote, link and chat posts. I'm working on video upload but it's disabled for now until I figure out why the uploaded videos break during transcoding. Features planned include (in no particular order)
  • Working video uploading
  • View the users dashboard (both on entries and those from people you follow)
  • Support for other character sets (Russian, Chinese etc)
  • Posting audio, unsure if there's any real use/need for this
It's an open source project under the MIT license so the source can be found here: github
